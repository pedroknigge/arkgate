# ArkGate 3.8.1

- **Released:** 2026-07-22
- **npm:** `arkgate@3.8.1` (`latest`)
- **Git tag:** `v3.8.1`
- **GitHub Release:** https://github.com/pedroknigge/arkgate/releases/tag/v3.8.1
- **Status:** published

ArkGate 3.8.1 is a backward-compatible corrective **patch** over 3.8.0. It closes silent
fail-open on peerIsolation when path/slice evidence is incomplete, improves pure-IR type-only
named imports and relative `require` edges, and lands the Domain analysis vocabulary pilot split
without changing public root/gate import paths.

## What changed

| Area | 3.8.1 result |
|------|--------------|
| peerIsolation (S1) | **Fail-closed:** missing paths, empty/unresolvable slice folders, or unclassifiable slices **deny** when isolation is configured — no silent fail-open |
| Pure IR type-only (S5) | All-type named lists (`import { type A }`) are type-only for capability evidence; default+named and mixed lists stay value |
| Relative require (S4) | Pure module graph emits edges for relative `require(...)`; package require remains capability-only |
| Domain pilot cluster | `analysis.ts` facade + `stableHash` + resolved-candidate-facts modules; public re-exports unchanged |
| Docs honesty | Pins and claim matrix align to 3.8.x; Z09 remains a parked claim gate (residual RB-11 only) |

## Upgrade

```bash
npm install -D arkgate@3.8.1
npx arkgate upgrade
```

**No required config migration.** Existing projects do not need a config change.

**Note:** if you use `peerIsolation: true` and some edges lack resolvable file paths (or slice
folders cannot classify either side), 3.8.1 may **deny** edges that 3.8.0 left open. That is the
safety fix. Same-slice edges with both paths classifiable still allow.

## Compatibility and limits

- No breaking CLI command, MCP tool, or required `ark.config.json` field change.
- Public `arkgate` / `arkgate/gate` import paths for analysis and resolved facts are unchanged.
- Lexical and single-snippet compatibility paths remain explicitly partial and non-green; strict
  CI is the final merge boundary.
- `@arkgate/runtime` is experimental, separately versioned, and not published by this release.
- Z09 and residual `RB-11` remain open for retained adoption and independently signed close. This
  release does not claim that Phase Z is shipped.
