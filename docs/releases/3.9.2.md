# ArkGate 3.9.2

- **Released:** 2026-07-24
- **npm:** `arkgate@3.9.2` (`dist-tags.latest`)
- **Git tag:** `v3.9.2`
- **GitHub Release:** https://github.com/pedroknigge/arkgate/releases/tag/v3.9.2
- **Status:** published

Patch over **3.9.1**. Product honesty: post-validity coaching, coverage/host write-path honesty, and
advisory analysis precision (graph-blind / ambient messaging). No required config migration.
Does not weaken write gate or CI. Does not promote Y07/Y09. Does not invent hard write on soft hosts.

## What changed

| Area | 3.9.2 result |
|------|--------------|
| Coverage honesty | Weak (&lt;50%) / partial / strong; green is not whole-tree enforcement until 100% |
| Baseline honesty | Dirty-freeze risk labeled when green-via-freeze hides material debt |
| Write-path honesty | Soft hosts fail-closed; hard hosts unverified without evidence |
| Design-weak coaching | Placement coaching; one pilot; multi-pilot + silent plan-B auto-apply forbidden |
| Graph blind spots | Advisory unresolvable dynamic edges (template-interpolation); never hard verdict |
| Ambient (Y07) | Status + parked-Y07 / `blockerGrade: false` honesty only |
| Skills | Deepen `ark-coverage` / `ark-explore` / `ark-place` (no new basenames) |

## What did not change

- Write gate / CI fail-closed semantics
- Host hardness claims (Claude/Grok/Antigravity hard when covered; Cursor/Codex/OpenCode advisory)
- Z09 / RB-11 field claims (still open)
- Y07 / Y09 ROADMAP status (still **parked**)
- Analysis hard verdicts (graph-blind is advisory only)

## Upgrade

```bash
npm install -D arkgate@3.9.2
# or: npm install -D arkgate@latest   # after this patch is latest
npx arkgate-check --install-agent-gates --force
npx arkgate-check --doctor --json
```

No config migration. Optional: refresh gates/skills after upgrade. Inspect doctor JSON fields
`coverageHonesty`, `writePath.honesty`, `graphBlindSpots`, `pilotLoop`, `ambientState`.
