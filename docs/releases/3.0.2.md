# ArkGate 3.0.2

**Released:** 2026-07-13  
**npm:** `arkgate@3.0.2` (`latest`)  
**Git tag:** `v3.0.2`  
**GitHub Release:** https://github.com/pedroknigge/arkgate/releases/tag/v3.0.2  
**Status:** published

Docs-only patch: ship the post-3.0.1 agent-guide / AI-gates / demo prose that documents
design-depth skills and JSON surfaces already in 3.0.1. **No runtime or CLI behavior change.**

## Upgrade

```bash
npm install -D arkgate@3.0.2
```

Skills were already current in 3.0.1; re-install only if you want template refresh:

```bash
npx arkgate-check --install-agent-gates --force
```
