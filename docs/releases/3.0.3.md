# ArkGate 3.0.3

**Released:** 2026-07-13  
**npm:** `arkgate@3.0.3` (`latest`)  
**Git tag:** `v3.0.3`  
**GitHub Release:** https://github.com/pedroknigge/arkgate/releases/tag/v3.0.3  
**Status:** published

Phase **Q** patch: power + simple surfaces for AI-clear work after ENFORCE.
**Additive** doctor/plan/MCP/eval fields and skill guidance. **No breaking** CLI or
`ark.config.json` changes. **No gate weaken.**

## Highlights

| Area | 3.0.3 result |
|------|----------------|
| **Q01 Post-green path** | When design-weak, doctor exposes `postGreenPath` / `primaryNextAction` (`clarify-for-ai`) — one Shape door, no skill shopping |
| **Q02 Smell outcomes** | Each `designSmells[]` has plain-language `outcome` (ids unchanged) |
| **Q03 Golden pattern** | Optional `.ark/golden-pattern.json`; place / prepare-write / doctor `goldenPattern` — advisory for **new** code only; never clears design-weak |
| **Q04 Pilot loop** | `plan.pilotLoop` / `doctor.pilotLoop` with **one** `nextPilot` extraction card → re-doctor; never mechanical-safe |
| **Q05 AI-velocity** | `npm run eval:ai-velocity` — fixture-measured `placementTurns` (golden-path strictly better than design-weak; no live LLM) |
| Skills | `/ark-place` golden; explore/autopilot pilot loop; `/ark-explain` opens `ark-report.html` in the default browser |

## Honesty (read this)

- Empty plan A + ENFORCE can still be **design-weak** — not “healthy finished.”
- `patternBets` / pilot loop remain **`neverMechanicalSafe: true`** — never auto-applied.
- Golden pattern is **optional** and **advisory**; absence is normal; presence does **not** ENFORCE or clear smells.
- AI-velocity is a **fixture proxy** (placement turns), not a live multi-agent productivity claim.

## Upgrade

```bash
npm install -D arkgate@3.0.3
npx arkgate-check --install-agent-gates --force
npx arkgate-check --doctor --json   # postGreenPath, designSmells[].outcome, goldenPattern, pilotLoop
npx arkgate-check --plan --json     # patternBets + pilotLoop
npm run eval:ai-velocity            # optional local / CI fixture bench
```

