# ArkGate 3.0.1

**Released:** 2026-07-13

**Git tag:** `v3.0.1`  
**GitHub Release:** https://github.com/pedroknigge/arkgate/releases/tag/v3.0.1

## Upgrade

```bash
npm install -D arkgate@3.0.1
npx arkgate-check --install-agent-gates --force
npx arkgate-check --doctor
npx arkgate-check --plan --json   # patternBets when design residual exists
```
